# cnode-electron
用electron 结合 react 制作的node中文社区客户端

使用方法
1 git clone https://github.com/wq123456/cnode-electron.git
----------------------------------------------------------
2 cd cnode-electron & npm install
----------------------------------
3 node server.js
----------------
4 打开浏览器 http://127.0.0.1:3000
---------------------------------
